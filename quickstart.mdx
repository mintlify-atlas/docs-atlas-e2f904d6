## Overview

This quickstart guide will help you build Muun Wallet from source and get it running on an Android device or emulator.

## Prerequisites

Before you begin, ensure you have:

- Git installed and the repository cloned
- Basic familiarity with Android development  
- An Android device or emulator for testing

## Local development build

### Requirements

- [OpenJDK](https://adoptopenjdk.net/?variant=openjdk11&jvmVariant=hotspot) >= 8
- [Golang](https://golang.org/dl/) = 1.24.x
- [Android NDK](https://developer.android.com/ndk/downloads) >= 22
- [Docker](https://docs.docker.com/engine/install/) >= 28

### Build steps

<Steps>
  <Step title="Set up environment">
    Set the `ANDROID_NDK_HOME` environment variable:
    
    ```bash
    export ANDROID_NDK_HOME=/path/to/your/ndk
    ```
  </Step>

  <Step title="Build Rust libraries">
    ```bash
    libwallet/librs/makelibs.sh
    ```
  </Step>

  <Step title="Bootstrap gomobile">
    ```bash
    tools/bootstrap-gomobile.sh
    ```
  </Step>

  <Step title="Build libwallet">
    ```bash
    tools/libwallet-android.sh
    ```
  </Step>

  <Step title="Build the APK">
    ```bash
    ./gradlew :android:apolloui:assembleProdRelease
    ```
  </Step>
</Steps>
